FAQ 帮助中心

您担忧的所有问题，我们都已想到
Q : 支付问题
A : 人民币，目前仅支持银行卡打款通道
Q : 订单确认
A : 银行转账时请确保正确填写转账备注，财务将在核实银行付款信息后，确认订单。
Q : 产品交付
A : 订单全额付款后，我方将在预计时间安排发货。物流服务由第三方提供，如德邦、顺丰。请注意，除非网站另有明确说明，否则产品的零售价格不包括运费，交货时间仅为预计时间。第三方服务商不对交货时间做任何保证，且不应依赖该等服务。 我方将产品交付承运人后，即完成交付，产品所有权移转至您，其毁损、灭失风险均由您承担。
Q : 退货退款
A : 本显卡矿机一经售出，不支持退款或退货。如有特殊情况下，请联系我们。
Q : 操作指导
A : 您可以通过本站提供的工作人员联系方式获取操作指导。
Q : 售后问题
A : 任何的售后维修问题请联系我们的工作人员。
Q : 瑕疵产品及产品问题导致的维修
A : 在质保期内，如果产品缺陷是因为产品本身的问题（不包括质保限制规定下的瑕疵）造成的，您可以按照质保规定自费退回商品进行维修，但应在发现产品瑕疵后的合理时间内退回产品。我方收到原产品并进过相关服务处理流程后，将重新发回经过维修的产品。您需要自己承担产品退回的运费。
Q : 电费结算
A : 我们将会以邮件的形式，于每月月初向您开通托管时提供的邮箱地址，发送上个月的电费账单和汇款信息，请于收到邮件后3个工作日内完成上个月电费的支付，否则矿场将会下架对应的矿机。对电费存在异议请直接回复电费邮件。
Q : 取消矿机托管
A : 1. 联系客服（微信 mayaminer002）结算电费
2. 填写取消托管信息登记表发送至 mailto:bo@maya.top（邮件标题为 姓名+取消托管
3. 点击下载“取消矿机托管登记表”
另：寄送矿机所产生的运费由用户自行承担
Q : 更改托管信息
A : 1. 填写托管信息更改表发送至 mailto:bo@maya.top（邮件标题为 姓名+更改操作+矿机编号）
2. 点击下载 “托管信息更改登记表”
另：寄送矿机所产生的运费由用户自行承担
Q : 矿机托管报修
A : 矿机托管后，请添加客服联系方式显示，客服将会帮您添加到对应的矿场群，若矿机出现停机、算力过低等异常情况，将矿机编号报修给矿场人员，矿场人员在收到报修请求后将会进行处理，处理完成时间取决于机器故障情况。提供托管服务的一方不承担因机器维修而导致的收益损失。
